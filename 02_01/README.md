# ДЗ1(2 семестр)

## Задача 1 

На C написать программу, которая отслеживает сигналы пришедшие этому процессу, и выводит в консоль:

"Signal <n> recieved."

## Задача 2 

Существует 2 процесса. У каждого процесса есть терминал. Создаём приглащение ко вводу.

Написали в один процесс. Нажали Enter. В другом процессе оказались на новой строке. + не надо терять свой ввод.

Делаем так и для наименованных и ненаименнованных pipe.

## Задача 3 

С помощью сокетов общаемся между 2 процессами по протоколу:

1. TCP
2. UDP
3. ICMP

Шифруем с помощью XOR. Также делаем терминал также как во 2 задании.
